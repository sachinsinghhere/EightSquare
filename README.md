## Getting Started ğŸ

1. Install dependencies:
```bash
yarn install
```

2. Install iOS dependencies (iOS only):
```bash
cd ios && bundle install && bundle exec pod install && cd ..
```

3. Start the development server:
```bash
yarn start
```

4. Run the app:
```bash
# For iOS
yarn ios

# For Android
yarn android
```
